Word Cheat Sheet

Shortcuts

Shortcut Description
CTRL + NCreate a new document.
CTRL + OOpen an existing document.
CTRL + SSave the current document.
CTRL + PPrint the document.
CTRL + ZUndo the last action.
CTRL + YRedo the last undone action.
CTRL + CCopy the selected text or object.
CTRL + XCut the selected text or object.
CTRL + VPaste the copied text or object.
CTRL + BApply bold formatting to selected text.
CTRL + IApply italic formatting to selected text.
CTRL + UApply underline formatting to selected text.
CTRL + ASelect all text and objects in the document.
CTRL + FFind text in the document.
CTRL + HReplace text in the document.
CTRL + KInsert a hyperlink.
CTRL + LAlign text to the left.
CTRL + EAlign text to the center.
CTRL + RAlign text to the right.
CTRL + JJustify text alignment.
