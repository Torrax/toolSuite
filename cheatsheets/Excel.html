Excel Cheat Sheet

Hotkeys

Hotkey Description
CTRL + FOpens the Find and Replace dialog box
CTRL + HOpens the Replace tab in the Find and Replace dialog box
CTRL + BApplies or removes bold formatting
CTRL + IApplies or removes italic formatting
CTRL + UApplies or removes underline formatting
CTRL + HOMEMoves to the beginning of the worksheet
CTRL + ENDMoves to the last cell of the worksheet
CTRL + Arrow KeysMoves to the edge of the data region
CTRL + SHIFT + LApplies or removes filters
CTRL + 1Opens the Format Cells dialog box
CTRL + TCreates a table from the selected data
CTRL + DFills the selected cell with the content above it
CTRL + RFills the selected cell with the content to its left
F2Edits the active cell
F4Repeats the last action
CTRL + ASelects the entire worksheet
CTRL + CCopies the selected cells
CTRL + VPastes the copied cells
CTRL + XCuts the selected cells
CTRL + ZUndoes the last action
CTRL + YRedoes the last action
CTRL + SSaves the current workbook
CTRL + POpens the print menu
ALT + EnterStarts a new line in the same cell
SHIFT + SpaceSelects the entire row
CTRL + SpaceSelects the entire column
CTRL + ALT + VOpens the Paste Special dialog box

Functions

Function Description
'=SUM(A1:A10)Adds the values in the range A1 to A10
'=AVERAGE(A1:A10)Calculates the average of the values in the range A1 to A10
'=IF(A1>10,"Yes","No")Returns 'Yes' if the value in A1 is greater than 10, otherwise returns 'No'
'=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])Searches for a value in the first column of a range and returns a value in the same row from another column
'=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])Searches for a value in the first row of a range and returns a value in the same column from another row
'=INDEX(array, row_num, [column_num])Returns the value of a cell in a specified row and column of a range
'=MATCH(lookup_value, lookup_array, [match_type])Returns the relative position of a value in a range
'=CONCAT(A1, " ", B1)Joins values in A1 and B1 with a space between them
'=LEN(A1)Counts the number of characters in a cell, including spaces
'=TRIM(A1)Removes all extra spaces from text except for single spaces between words
'=LEFT(A1, 5)Extracts the first 5 characters from the left side of the text in A1
'=RIGHT(A1, 5)Extracts the last 5 characters from the text in A1
'=MID(A1, 2, 3)Extracts 3 characters starting from the 2nd character in A1
'=TEXT(A1, "mm/dd/yyyy")Converts a number to a specified date format
'=NOW()Returns the current date and time
'=TODAY()Returns the current date
'=ROUND(A1, 2)Rounds the value in A1 to 2 decimal places
'=ROUNDUP(A1, 2)Rounds the value in A1 up to 2 decimal places
'=ROUNDDOWN(A1, 2)Rounds the value in A1 down to 2 decimal places
'=COUNT(A1:A10)Counts the number of numeric entries in the range A1 to A10
'=COUNTA(A1:A10)Counts the number of non-empty cells in the range A1 to A10
'=COUNTIF(A1:A10,">10")Counts the number of cells in the range A1 to A10 that are greater than 10
'=PMT(rate, nper, pv, [fv], [type])Calculates the payment for a loan based on constant interest rates and payments
'=PROPER(A1)Capitalizes the first letter of each word in A1
'=UPPER(A1)Converts text in A1 to uppercase
'=LOWER(A1)Converts text in A1 to lowercase
'=TEXTJOIN(delimiter, ignore_empty, text1, [text2, ...])Joins text strings with a specified delimiter, optionally ignoring empty cells.
'=SUBSTITUTE(text, old_text, new_text, [instance_num])Replaces occurrences of old text with new text in a string.
'=DATEDIF(start_date, end_date, unit)Calculates the difference between two dates in days, months, or years.
'=WORKDAY(start_date, days, [holidays])Returns the date after a specified number of workdays, excluding weekends and holidays.
'=AND(logical1, [logical2, ...])Returns TRUE if all the arguments evaluate to TRUE.
'=OR(logical1, [logical2, ...])Returns TRUE if any of the arguments evaluate to TRUE.
'=NOT(logical)Reverses the logical value (TRUE becomes FALSE, and vice versa).
