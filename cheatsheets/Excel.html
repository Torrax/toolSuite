Excel Cheat Sheet

Hotkeys

Hotkey Description
Editing
F2 Edit the active cell.
CTRL+C Copy selected cells.
CTRL+V Paste copied cells.
CTRL+X Cut selected cells.
CTRL+Z Undo last action.
CTRL+Y Redo last action.
Formatting
CTRL+B Bold formatting.
CTRL+I Italic formatting.
CTRL+U Underline formatting.
Navigation
CTRL+HOME Go to the beginning of the worksheet.
CTRL+END Go to the last cell with content.
CTRL+Arrow Keys Move to the edge of the data region.
Selection
CTRL+A Select the entire worksheet.
SHIFT+Space Select the entire row.
CTRL+Space Select the entire column.
Miscellaneous
CTRL+S Save the current workbook.
CTRL+P Open the print menu.
ALT+ENTER Start a new line in the same cell.
CTRL+F Open Find dialog box.
CTRL+H Open Replace dialog box.

Functions

Function Description
Math Functions 
=SUM(A1:A10)
 Adds the values in the range A1 to A10. 
=AVERAGE(A1:A10)
 Calculates the average of the values in the range A1 to A10. 
=ROUND(A1, 2)
 Rounds the value in A1 to 2 decimal places. 
=PRODUCT(A1:A5)
 Multiplies all the numbers in the range A1 to A5. 
=INT(A1)
 Rounds the number in A1 down to the nearest integer. 
=MOD(A1, 2)
 Returns the remainder after dividing A1 by 2. 
=POWER(A1, 3)
 Raises the value in A1 to the power of 3. 
=SQRT(A1)
 Calculates the square root of the value in A1.
Logical Functions 
=IF(A1>10,"Yes","No")
 Returns 'Yes' if A1 > 10, otherwise 'No'. 
=AND(A1>0, B1<5)
 Returns TRUE if both conditions are TRUE. 
=OR(A1>10, B1>10)
 Returns TRUE if at least one condition is TRUE. 
=NOT(A1>10)
 Reverses the logical value of A1 > 10. 
=IFERROR(A1/B1, "Error")
 Returns "Error" if A1/B1 results in an error.
Text Functions 
=CONCAT(A1, " ", B1)
 Joins values in A1 and B1 with a space. 
=LEN(A1)
 Counts the number of characters in A1. 
=LEFT(A1, 3)
 Extracts the first 3 characters from A1. 
=RIGHT(A1, 2)
 Extracts the last 2 characters from A1. 
=MID(A1, 2, 3)
 Extracts 3 characters from A1 starting at position 2. 
=TRIM(A1)
 Removes extra spaces from A1. 
=SUBSTITUTE(A1, "old", "new")
 Replaces "old" with "new" in A1.
Date & Time Functions 
=NOW()
 Returns the current date and time. 
=TODAY()
 Returns the current date. 
=DATE(2024, 12, 25)
 Creates a date for December 25, 2024. 
=TIME(14, 30, 0)
 Creates a time value for 2:30 PM. 
=YEAR(A1)
 Extracts the year from the date in A1. 
=MONTH(A1)
 Extracts the month from the date in A1. 
=DAY(A1)
 Extracts the day from the date in A1. 
=HOUR(A1)
 Extracts the hour from the time in A1. 
=MINUTE(A1)
 Extracts the minute from the time in A1. 
=SECOND(A1)
 Extracts the second from the time in A1.
Lookup & Reference Functions 
=VLOOKUP(A1, B1:C10, 2, FALSE)
 Searches for A1 in the first column and returns a value from the second column. 
=HLOOKUP(A1, B1:C10, 2, FALSE)
 Searches for A1 in the first row and returns a value from the second row. 
=XLOOKUP(A1, B1:B10, C1:C10, "Not Found")
 Searches for A1 in B1:B10 and returns the corresponding value from C1:C10 or "Not Found". 
=MATCH(A1, B1:B10, 0)
 Finds the position of A1 within the range B1:B10. 
=INDEX(C1:C10, MATCH(A1, B1:B10, 0))
 Returns the value from C1:C10 that corresponds to the position of A1 in B1:B10. 
=OFFSET(A1, 2, 3)
 Returns the cell that is 2 rows down and 3 columns to the right of A1. 
=CHOOSE(2, "Apple", "Banana", "Cherry")
 Returns "Banana" based on the index 2.
Financial Functions 
=PMT(0.05/12, 60, 10000)
 Calculates the monthly payment for a loan with a 5% annual interest rate, 60 periods, and a present value of 10,000. 
=FV(0.05/12, 60, -200, -10000)
 Calculates the future value of an investment with a monthly payment of 200, 60 periods, and a present value of 10,000.
Statistical Functions 
=COUNT(A1:A10)
 Counts the number of numeric entries in A1:A10. 
=COUNTA(A1:A10)
 Counts the number of non-empty cells in A1:A10. 
=COUNTIF(A1:A10, ">5")
 Counts the number of cells in A1:A10 that are greater than 5. 
=AVERAGEIF(A1:A10, ">5", B1:B10)
 Calculates the average of B1:B10 where A1:A10 is greater than 5. 
=MEDIAN(A1:A10)
 Finds the median value in A1:A10. 
=MODE(A1:A10)
 Finds the most frequently occurring value in A1:A10. 
=STDEV(A1:A10)
 Calculates the standard deviation of the values in A1:A10.
Information Functions 
=ISNUMBER(A1)
 Checks if A1 contains a number. 
=ISTEXT(A1)
 Checks if A1 contains text.
Lookup & Reference Functions (Continued) 
=MATCH(A1, B1:B10, 0)
 Finds the position of A1 within the range B1:B10. 
=INDEX(C1:C10, MATCH(A1, B1:B10, 0))
 Returns the value from C1:C10 that corresponds to the position of A1 in B1:B10. 
=OFFSET(A1, 2, 3)
 Returns the cell that is 2 rows down and 3 columns to the right of A1. 
=CHOOSE(2, "Apple", "Banana", "Cherry")
 Returns "Banana" based on the index 2.
Financial Functions (Continued) 
=NPER(0.05/12, -200, 10000)
 Calculates the number of periods for a loan with a 5% annual interest rate, monthly payment of 200, and present value of 10,000. 
=RATE(60, -200, 10000)
 Calculates the interest rate per period for a loan with 60 periods, monthly payment of 200, and present value of 10,000.
