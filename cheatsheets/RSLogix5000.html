RSLogix 5000 Cheat Sheet

Hotkeys

Hotkey Description
CTRL + E Opens the Edit Ladder view.
CTRL + G Go to a specific rung in the ladder logic.
CTRL + R Verify the rung logic for errors.
F4 Toggle force enable on a selected tag.
ALT + F12 Access cross-referencing for the selected element.
CTRL + F Search within the project file.
CTRL + SHIFT + F Open the Find All References window.
CTRL + D Download the program to the PLC.
CTRL + SHIFT + D Download with edits only.
F7 Compile the ladder logic.
CTRL + W Close the current project window.
CTRL + S Save the current project.

Functions

Function Description
ADD Adds two values together.
SUB Subtracts one value from another.
MUL Multiplies two values.
DIV Divides one value by another.
TON Timer On Delay function block.
TOF Timer Off Delay function block.
CTU Count Up function block.
CTD Count Down function block.
RTO Retentive Timer function block.
OSR One-Shot Rising function block.
OSF One-Shot Falling function block.
MOV Move data from one location to another.
LEN Get the length of a string.
ABS Returns the absolute value of a number.
