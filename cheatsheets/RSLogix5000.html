RSLogix 5000 Cheat Sheet

Hotkeys

Hotkey Description
CTRL + EOpens the Edit Ladder view.
CTRL + GGo to a specific rung in the ladder logic.
CTRL + RVerify the rung logic for errors.
F4Toggle force enable on a selected tag.
ALT + F12Access cross-referencing for the selected element.
CTRL + FSearch within the project file.
CTRL + SHIFT + FOpen the Find All References window.
CTRL + DDownload the program to the PLC.
CTRL + SHIFT + DDownload with edits only.
F7Compile the ladder logic.
CTRL + WClose the current project window.
CTRL + SSave the current project.

Functions

Function Description
ADDAdds two values together.
SUBSubtracts one value from another.
MULMultiplies two values.
DIVDivides one value by another.
TONTimer On Delay function block.
TOFTimer Off Delay function block.
CTUCount Up function block.
CTDCount Down function block.
RTORetentive Timer function block.
OSROne-Shot Rising function block.
OSFOne-Shot Falling function block.
MOVMove data from one location to another.
LENGet the length of a string.
ABSReturns the absolute value of a number.
