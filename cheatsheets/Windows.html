Windows Cheat Sheet

Hotkeys

Shortcut Description
WIN + DShow desktop
WIN + EOpen File Explorer
WIN + LLock your PC
WIN + TabOpen Task View
ALT + TabSwitch between open apps
CTRL + CCopy selected item(s)
CTRL + XCut selected item(s)
CTRL + VPaste item(s)
CTRL + ZUndo an action
CTRL + YRedo an action
SHIFT + DELETEPermanently delete an item
CTRL + ASelect all items in a window or on the desktop
ALT + ENTEROpen properties for a selected item
CTRL + SHIFT + ESCOpen Task Manager
WIN + ROpen the Run dialog box
WIN + IOpen the Settings app
WIN + XOpen the Quick Link menu
WIN + PauseOpen System properties
WIN + Up ArrowMaximize the active window
WIN + Down ArrowMinimize the active window
WIN + Left ArrowSnap the active window to the left half of the screen
WIN + Right ArrowSnap the active window to the right half of the screen
ALT + F4Close the active window
WIN + SHIFT + STake a screenshot using Snip & Sketch
WIN + Plus (+)Open Magnifier and zoom in
WIN + Minus (-)Zoom out using Magnifier
WIN + UOpen the Ease of Access Center
CTRL + ALT + DELOpen Security Options (lock, switch user, etc.)
WIN + VOpen the clipboard history
WIN + PProject a screen (e.g., extend or duplicate displays)
WIN + SpaceSwitch between input languages or keyboard layouts
WIN + NumberOpen the app pinned to the taskbar at the corresponding position
ALT + Up ArrowGo up one folder level in File Explorer
ALT + Left ArrowGo back to the previous folder
ALT + Right ArrowGo forward to the next folder
F2Rename the selected file or folder
F3Open the search box in File Explorer
WIN + CTRL + DCreate a new virtual desktop
WIN + CTRL + Left/Right ArrowSwitch between virtual desktops
WIN + CTRL + F4Close the current virtual desktop
WIN + CTRL + Shift + BRestart your display drivers (useful if the screen freezes)
WIN + GOpen the Game Bar (for screen recording and performance monitoring)
WIN + HOpen the voice typing tool
WIN + KOpen the Connect menu for wireless displays and audio devices
CTRL + Shift + EnterOpen Command Prompt/PowerShell as an administrator
CTRL + CCopy the selected text in Command Prompt
CTRL + VPaste the text into Command Prompt
WIN + .Open the emoji picker
WIN + ;Open the emoji picker (same as WIN + .)
WIN + +Zoom in using Magnifier
WIN + -Zoom out using Magnifier
WIN + CTRL + EnterTurn on Narrator
WIN + CTRL + NOpen Narrator settings
CTRL + Scroll WheelZoom in or out on the desktop or a document
CTRL + Shift + TReopen the last closed tab in most browsers
ALT + DSelect the address bar in File Explorer or a browser
CTRL + Shift + EscOpen Task Manager directly
WIN + Shift + MRestore minimized windows
WIN + NumberLaunch apps pinned to the taskbar (based on their position)
CTRL + ALT + Arrow KeysRotate the display (useful for multi-monitor setups)
