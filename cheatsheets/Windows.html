Windows Shortcuts Cheat Sheet

Keyboard Shortcuts

Shortcut Description
WIN + D Show desktop.
WIN + E Open File Explorer.
WIN + L Lock your PC.
WIN + Open Task View.
ALT + Switch between open apps.
CTRL + C Copy selected item(s).
CTRL + X Cut selected item(s).
CTRL + V Paste item(s).
CTRL + Z Undo an action.
CTRL + Y Redo an action.
SHIFT + DELETE Permanently delete an item.
CTRL + A Select all items in a window or on the desktop.
ALT + ENTER Open properties for a selected item.
CTRL + SHIFT + ESC Open Task Manager.
WIN + R Open the Run dialog box.
WIN + I Open the Settings app.
WIN + X Open the Quick Link menu.
WIN + Pause Open System properties.
WIN + Maximize the active window.
WIN + Minimize the active window.
WIN + Snap the active window to the left half of the screen.
WIN + Snap the active window to the right half of the screen.
ALT + F4 Close the active window.
WIN + SHIFT + S Take a screenshot using Snip & Sketch.
WIN + + Zoom in using Magnifier.
WIN + - Zoom out using Magnifier.
WIN + U Open the Ease of Access Center.
CTRL + ALT + DEL Open Security Options (lock, switch user, etc.).
WIN + V Open the clipboard history.
WIN + P Project a screen (e.g., extend or duplicate displays).
WIN + Space Switch between input languages or keyboard layouts.
WIN + Number Open the app pinned to the taskbar at the corresponding position.
ALT + Go up one folder level in File Explorer.
ALT + Go back to the previous folder.
ALT + Go forward to the next folder.
F2 Rename the selected file or folder.
F3 Open the search box in File Explorer.
WIN + CTRL + D Create a new virtual desktop.
WIN + CTRL + / Switch between virtual desktops.
WIN + CTRL + F4 Close the current virtual desktop.
WIN + CTRL + SHIFT + B Restart your display drivers (useful if the screen freezes).
WIN + G Open the Game Bar (for screen recording and performance monitoring).
WIN + H Open the voice typing tool.
WIN + K Open the Connect menu for wireless displays and audio devices.
CTRL + SHIFT + Enter Open Command Prompt/PowerShell as an administrator.
CTRL + C Copy the selected text in Command Prompt.
CTRL + V Paste the text into Command Prompt.
WIN + . Open the emoji picker.
WIN + ; Open the emoji picker (same as WIN + .).
WIN + + Zoom in using Magnifier.
WIN + - Zoom out using Magnifier.
WIN + CTRL + Enter Turn on Narrator.
WIN + CTRL + N Open Narrator settings.
CTRL + Scroll Wheel Zoom in or out on the desktop or a document.
CTRL + SHIFT + T Reopen the last closed tab in most browsers.
ALT + D Select the address bar in File Explorer or a browser.
CTRL + SHIFT + Esc Open Task Manager directly.
WIN + Shift + M Restore minimized windows.
CTRL + ALT + / / / Rotate the display (useful for multi-monitor setups).
