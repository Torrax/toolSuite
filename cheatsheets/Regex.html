Regex Cheat Sheet

Patterns

Pattern Description Example
.Matches any character except a newline.a.b matches 'acb', 'a1b', but not 'ab'.
^Anchors the match at the start of a line.^abc matches 'abc' at the beginning of a string.
$Anchors the match at the end of a line.abc$ matches 'abc' at the end of a string.
*Matches 0 or more of the preceding character.ab*c matches 'ac', 'abc', 'abbbc'.
+Matches 1 or more of the preceding character.ab+c matches 'abc', 'abbbc', but not 'ac'.
?Matches 0 or 1 of the preceding character.ab?c matches 'ac' or 'abc', but not 'abbc'.
{n}Matches exactly n instances of the preceding character.a{3} matches 'aaa', but not 'aa'.
{n,}Matches n or more instances of the preceding character.a{2,} matches 'aa', 'aaa', 'aaaa', etc.
{n,m}Matches between n and m instances.a{2,4} matches 'aa', 'aaa', or 'aaaa'.
[abc]Matches any character inside the brackets.[abc] matches 'a', 'b', or 'c'.
[^abc]Matches any character not in the brackets.[^abc] matches any character except 'a', 'b', or 'c'.
\dMatches any digit (0-9).\d matches '1', '2', etc.
\DMatches any non-digit.\D matches 'a', '@', etc.
\wMatches any word character (letters, digits, underscores).\w matches 'a', '1', '_'.
\WMatches any non-word character.\W matches '@', ' ', etc.
\sMatches any whitespace character.\s matches ' ', '\t', '\n'.
\SMatches any non-whitespace character.\S matches 'a', '1', '@'.
