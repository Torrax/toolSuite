CMD Cheat Sheet

Commands

Command Description Example
cdChange the directory.cd C:\Users\User
dirList the contents of a directory.dir
clsClear the Command Prompt screen.cls
exitClose the Command Prompt.exit
echoDisplay messages or enable/disable command echoing.echo Hello, World!
copyCopy files from one location to another.copy file.txt D:\backup
moveMove or rename files.move file.txt newfile.txt
delDelete files.del file.txt
mkdirCreate a new directory.mkdir new_folder
rmdirRemove a directory.rmdir /s folder_name
ipconfigDisplay network configuration.ipconfig /all
pingTest connectivity to a network host.ping google.com
hostnameDisplay the computer's hostname.hostname
systeminfoDisplay detailed system information.systeminfo
tasklistList all running processes.tasklist
taskkillTerminate a process by name or PID.taskkill /IM notepad.exe /F
chkdskCheck and repair disk errors.chkdsk C: /f
diskpartManage disk partitions.diskpart
formatFormat a disk.format E:
fsutilDisplay or manage filesystem properties.fsutil dirty query C:
attribDisplay or change file attributes.attrib +h file.txt
netstatDisplay network connections and statistics.netstat -an
tracertTrace the route to a network host.tracert google.com
net useMap a network drive.net use Z: \\server\share
net userManage user accounts.net user username password /add
setDisplay or set environment variables.set PATH=C:\NewPath;%PATH%
assocView or change file associations.assoc .txt=txtfile
fcCompare two files.fc file1.txt file2.txt
treeDisplay a graphical representation of a directory structure.tree
shutdownShutdown or restart the computer.shutdown /r /t 30 (restart in 30 seconds)
helpDisplay help for CMD commands.help copy
titleSet the Command Prompt window title.title My CMD Session
colorChange the text and background color.color 1F
