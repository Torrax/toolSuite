Linux Cheat Sheet

Commands

Command Description Example
lsList files and directories.ls -l (detailed view)
cdChange directory.cd /path/to/directory
pwdPrint the current directory.pwd
mkdirCreate a new directory.mkdir new_directory
rmRemove files or directories.rm file_name; rm -r directory_name
cpCopy files or directories.cp file1 file2
mvMove or rename files.mv old_name new_name
chmodChange file permissions.chmod 755 file_name
chownChange file ownership.chown user:group file_name
findSearch for files in a directory hierarchy.find /path -name "file_name"
grepSearch text using patterns.grep "pattern" file_name
psDisplay currently running processes.ps aux
killTerminate a process by PID.kill -9 PID
topDisplay real-time system activity.top
dfShow disk usage of file systems.df -h
duEstimate file or directory space usage.du -sh directory_name
tarCreate or extract tar archives.tar -cvf archive.tar file_name
zipCompress files.zip archive.zip file_name
unzipExtract compressed files.unzip archive.zip
wgetDownload files from the web.wget http://example.com/file
curlTransfer data from or to a server.curl http://example.com
sshConnect to a remote server securely.ssh user@hostname
scpCopy files over SSH.scp file_name user@hostname:/path
manDisplay the manual for a command.man ls
aliasCreate shortcuts for commands.alias ll='ls -la'
